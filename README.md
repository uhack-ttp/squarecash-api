# SQUARECASH API

SquareCash API is an API-only Application for SquareCash.

### Dependencies:
* Ruby 2.3.1 or later
* Rails 5.1.4

### Getting Started:

Install your gems
`bundle install`


Modify `database.yml` in accordance to your credentials.

Create your database:
```
$ bin/rake db:create
```

Run the database migrations:
```
$ bin/rake db:migrate
```

That's it! You're good to go!
```
$ bin/rails server
```
